Patient Study Calendar: Public demo site

Testing suggestions

Thank you for testing the Patient Study Calendar. The software you are about to test is PSC 2.4.1, released 1/16/2009.

Two considerations before you begin:

  1. This is a public instance. You may encounter sample study calendars created by other users. Other users will be able to see data that you enter. Please do not enter any confidential information.
  2. With each release, we may delete information from our database in order to prevent the site from getting cluttered with incomplete templates and abandoned calendars.

If you are new to PSC, we recommend that you begin with the following resources:

  1. Application Overview (in Power Point format)
  2. End User Guide
  3. Template Creation Guide
  4. Training Module (requires Flash)

To begin testing PSC:

  1. Login using superuser as both the username and password.
  2. You will have access to all roles and data within PSC.
  3. Please post feedback or reports of any problems that you encounter to the caBIG Knowledge Center Forums.

Let the testing begin: Public Test Site - start page

For organizations that are adopting PSC, we are making available a list of activities and some sample templates.

To import the activities into your instance of PSC:

  1. Download the XML file here.
  2. While logged in to PSC as a Study Coordinator, create or edit a template and click on a period in order to add activities to it.
  3. Click the "Import activities from xml" link.
  4. Browse to the XML file that you downloaded in step 1 and click "Import."

To import a template into your instance of PSC:

  1. Download one of the template files listed below.
  2. While logged in to PSC as a Study Coordinator, click the "Calendars" tab.
  3. Click the "Import an existing template" link.
  4. Browse to the XML file that you downloaded in step 1 and click "Import."
  5. You may now modify the template to suit your needs, add activities, and release the template for use.

Available templates:

  1. Completed template comparing radiation to radiation plus investigational treatment
  2. Uncompleted template comparing two treatment regimens
  3. Uncompleted template involving a transplant
  4. Uncompleted template that watches for certain outcomes during treatment
